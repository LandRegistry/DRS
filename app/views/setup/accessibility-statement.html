Accessibility statement

This website is run by HM Land Registry. We want as many people as possible to be able to use it. For example, that means you should be able to:

  • change colours, contrast levels and fonts
  • zoom in up to 200% without the text spilling off the screen
  • navigate most of the website using just a keyboard
  • navigate most of the website using speech recognition software
  • listen to most of the website using a screen reader (including the most recent versions of JAWS, NVDA and VoiceOver)

We’ve also made the website text as simple as possible to understand.

AbilityNet has advice on making your device easier to use if you have a disability.

How accessible is this website?

We know some parts of this website aren’t fully accessible. For example:

  • it’s not always possible to return to the homepage
  • some headings are clipped at high magnification sizes
  • some button colours are not consistent
  • some link highlighting is not consistent
  • some radio buttons don’t behave as intended

What to do if you can’t access parts of this website

If you need information on this website in a different format like accessible PDF, large print, easy read, audio recording or braille, make a request using these details:

  • phone on 0300 006 0411
  • post at HM Land Registry Citizen Centre, PO Box 74, Gloucester, GL14 9BB
  • online

We’ll consider your request and get back to you within 5 days.

Reporting accessibility problems with this website

We’re always trying to improve the accessibility of this website. If you find any problems that aren’t listed on this page or if you think we’re not meeting accessibility standards, contact the Customer Services Centre by:

  • phone on 0300 006 0411
  • post at HM Land Registry Citizen Centre, PO Box 74, Gloucester, GL14 9BB
  • online

Enforcement procedure

The Equality and Human Rights Commission (EHRC) is responsible for enforcing the Public Sector Bodies (Websites and Mobile Applications) (No. 2) Accessibility Regulations 2018 (the ‘accessibility regulations’). If you’re not happy with how we respond to your complaint, contact the Equality Advisory and Support Service (EASS).

Contact us by phone or visit us in person

We provide a text relay service for people who are D/deaf, hearing impaired or have a speech impediment.

Our offices have audio induction loops. Contact us before your visit if you need a British Sign Language (BSL) interpreter.

Technical information about this website’s accessibility

HM Land Registry is committed to making its website accessible, in accordance with the Public Sector Bodies (Websites and Mobile Applications) (No. 2) Accessibility Regulations 2018.

This website is partially compliant with the Web Content Accessibility Guidelines version 2.1 AA standard.

Non-accessible content

The content listed below is non-accessible for the following reasons.

Non-compliance with the accessibility regulations

  • Users cannot return to the start of the journey once they have begun. This fails WCAG 2.1 AA criterion 2.4.5 (Multiple Ways).

  • Some headings appear clipped. This fails WCAG 2.1 AA criterion 1.4.10 (Reflow).

  • The red delete button sometimes appears green when tabbing through the page. This fails WCAG 2.1 AA criterion 3.2.1 (On Focus).

  • The highlight colour shown when using the tab key to navigate through links is sometimes inconsistent. This fails WCAG 2.1 AA criterion 2.4.7 (Focus Visible).

  • Some parts of the ‘Select lender representation’ function will not be visible to screen readers. This fails WCAG 2.1 A criterion 4.1.1 (Validate the HTML).

  • Some selected radio buttons do not remain selected when returned to. This fails WCAG 2.1 AA criterion 3.2.3 (Consistent Navigation).

  • In a few instances, users are not informed that links will open in a new window or tab. This fails WCAG 2.1 AA criterion 2.4.4 (Link Purpose).

  • Some tables do not have captions to help those using assistive technology. This fails WCAG 2.1 AA criterion 1.3.2 (Meaningful Sequence).

Disproportionate burden

It is not considered that any aspect of HM Land Registry’s services will fall into this category. Areas of non-compliance with the guidelines are detailed in other sections of this statement.

Content that’s not within the scope of the accessibility regulations

PDFs and other documents

The accessibility regulations do not require us to fix PDFs or other documents published before 23 September 2018 if they’re not essential to providing our services. For example, we do not plan to fix Transfer under a chargee’s power of sale (PG75).

How we tested this website

This website was last tested on 1 May 2020. Tests were carried out by an independent internal audit team using a purpose-built test suite. We aim to fix the reported defects by 30 September 2020.

What we’re doing to improve accessibility

We've written this statement based on the defects found on 1 May 2020. As we are still building this service, we'll continue to test it and all new pages. We'll update this statement as we find or fix any defects.
