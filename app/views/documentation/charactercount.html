Character count

Character counts are used in the service where an input has a limit which could regularly be exceeded by the user.

The service currently only allows user references of 25 characters or fewer - this is due to database and legacy systems.

We've previously used character count on large text fields abd single line fields, however, due to an update to the GDS toolkit, we can no longer use the character count component on single text fields - without additional technical work to implement.

User research

There has been a limited amount of feedback around character limits - where the character count is not used, some users are unhappy they are able to enter information and then face a frontend error, these users would rather be told before they've clicked continue.
