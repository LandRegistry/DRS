Using the prototype

The prototype is primarily for user research sessions. Not all functionality will work and unexpected errors are likely to occur.

When using the prototype, some pages will feature dynamic data, whilst others have set information for use in testing scenarios or to enable rapid prototyping.

Simple, happy paths are more likely to work end-to-end.

Specific journeys with shortcuts to different stages of the service can be found on the home page.

For support using this prototype, contact Ros Boyes or Kristin Hall.
